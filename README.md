## ✨ Функциональность

- 📥 Добавление RSS-фидов по URL
- 🔄 Автоматическое обновление постов каждые 5 секунд
- 👁 Предпросмотр постов в модальном окне
- ✅ Отметка прочитанных постов
- 🛡 Валидация формы с понятными сообщениями об ошибках
- 📚 Поддержка нескольких фидов одновременно
